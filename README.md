🎯 Event Management System

📌 Project Overview

The Event Management System is a web-based application designed to simplify the process of organizing and managing events such as workshops, competitions, and seminars.

The system allows administrators to create, edit, and delete events, while participants can browse available events and register for them.

---

📝 Project Description

A web-based system for managing events and workshops. The system allows the admin to create, edit, and delete events, while participants can view available events and register for them.
