# 🎯 Event Management System

## 📌 Project Overview

The Event Management System is a web-based application designed to simplify the process of organizing and managing events such as workshops, competitions, and seminars.  
It allows administrators to create, edit, and manage events, while participants can browse available events and register بسهولة.

---

## 🏗️ Tech Stack

- ASP.NET Core MVC  
- SQL Server  
- Entity Framework Core  
- GitHub

## 🧩 System Features

- Create new events  
- Edit existing events  
- Delete events  
- View all events  
- Register for events  
- Store participant data  
- Admin dashboard for viewing registrations  

---

## 👤 User Roles

### 🔹 Admin
- Create, edit, and delete events  
- Manage events  
- View registered participants  

### 🔹 Participant
- View available events  
- Register for events
