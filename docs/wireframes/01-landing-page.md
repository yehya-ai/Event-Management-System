# Landing Page Wireframe

## Overview

The landing page is the main entry point of the system.  
It provides a quick overview of the platform and allows users to explore available events.

---

## Main Sections

- Navigation Bar  
- Hero Section  
- Events Section  
- Footer  

---

## UI Elements

### Navigation Bar
- Logo (Event Management System)
- Links:
  - Home
  - Events
  - Login

### Hero Section
- Title: "Manage Your Events بسهولة"
- Short description of the system
- Button: "View Events"

### Events Section
- List of upcoming events
- Each event includes:
  - Event title
  - Date
  - Short description
  - "View Details" button

### Footer
- Basic information
- Contact placeholder

---

## User Actions

- User can browse events
- User can click on event details
- User can navigate to login page

---

## Notes

- The design should be simple and clean  
- Focus on easy navigation  
- Highlight important actions like "View Events"
