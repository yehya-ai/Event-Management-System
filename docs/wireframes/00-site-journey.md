# Site Journey

## Overview

This file describes the main navigation flow of the Event Management System for different types of users.

---

## Guest User Path

Home Page → View Events → View Event Details → Register

### Description
- The guest user starts from the Home Page
- The user can browse available events
- The user can open event details
- The user can register for an event without creating an account

---

## Logged-in Admin Path

Login Page → Admin Dashboard → Manage Events → View Participants

### Description
- The admin starts from the Login Page
- After login, the admin accesses the Dashboard
- The admin can create, edit, or delete events
- The admin can view registered participants for each event
