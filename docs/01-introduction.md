# Introduction

## Problem Statement
Managing events manually using Excel or messaging apps leads to errors, poor organization, and wasted time.

## Objectives
- Simplify event management
- Reduce manual work
- Provide centralized system
- Improve data organization
- Provide statistics for admin

## Scope
The system allows admins to manage events and users to register.

## Constraints
- Limited time
- Small team
- Basic system version
